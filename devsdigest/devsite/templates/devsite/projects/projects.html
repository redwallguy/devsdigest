Milton: the Discord soundboard

Milton is my pride and joy; my greatest creation. It represents the cumulative expertise of my programming skills, growing in capability and utility as I did the same.

Milton is a soundboard bot for the popular VoIP service, Discord. It allows users in its Discord servers to play sound clips on command, set clips to play as they join a voice channel, and display the available clips as desired. Admin users can use its website couterpart to upload new clips and organize them, grouping them into "boards" of clips. They can even give clips with unwieldy, long names an alias, so as to call them with ease.

Milton is quite extensive, and showcases a wide knowledge of languages, practices, and frameworks, including:

  • Languages:
    • Python
    • Javascript
  • Frameworks/Libraries:
    • Django
    • jQuery
  • Practices:
    • Utilizing public APIs
    • Asynchronous programming
    • Rate limiting
    • Task scheduling
  • Services:
    • AWS S3 cloud storage
    • Heroku deployment

The Digest

My first ever personal programming project, this endeavor led me down the path of programming. Me and my friends were half-complaining about having to go to each individual community on Reddit every day to find the best content in each one every day. I thought that there must be a bot to do it already made, but I couldn't find one. So I decided to simply do it myself.

I started tinkering with Python and figuring out how to do more and more until I finally had an amalgamation of tools that could cobble together a working bot. The feeling when I saw it actually working was incredible, so I continued programming and never stopped.

As far as first projects go, this has been surprisingly durable, and works to this day. It uses:

  • Languages:
    • Python
  • Libraries:
    • PRAW (Reddit API wrapper)
    • imgurpython (Imgur API wrapper)
    • Discord.py (Discord API wrapper)

The source code for the digest can be found here.
